Detailed Description

Calduino data type struct definition. Each data type contains a name, an encode type, a calduino unit and information to parse the value from the EMS Datagram (offset, bitOffset, floatBytes and floatFactor). Offset is a mandatory parameter. Bit Offset should be only used when encode type is Bit. Float bytes and factor are only used in float data types (float numbers can occupy one or two bytes and come multiplied by two or ten to avoid decimals).
